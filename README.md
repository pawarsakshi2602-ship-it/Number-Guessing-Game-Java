#  Number Guessing Game (Java Project)

##  Project Description

This is a simple console-based Number Guessing Game developed using Java. The system generates a random number and the user has to guess it within limited attempts.

##  Features

* Random number generation (1–100)
* User input for guessing
* Hints (Too High / Too Low)
* Limited attempts
* Score calculation
* Multiple rounds option

##  Technologies Used

* Java
* OOP Concepts
* Eclipse IDE

##  Project Structure

* `NumberGuessingGame.java` → Main class containing game logic

##  How to Run

1. Open project in Eclipse
2. Run `NumberGuessingGame.java`
3. Enter your guesses
4. Follow instructions on screen
